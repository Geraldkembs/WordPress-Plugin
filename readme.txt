A simple and flexible social media icons manager with drag and drop ordering.

== Description ==

Social Media Manager allows you to easily add and manage social media icons on your WordPress website. Features include:

* Drag and drop reordering of social media icons
* Enable/disable individual social media platforms
* Choose between header and footer display
* Customizable social media URLs
* Responsive design
* Shortcode support [social_media_icons]
* Clean and modern design

Supported Social Media Platforms:
* Facebook
* Twitter
* Instagram
* LinkedIn
* YouTube
* WhatsApp

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/social-media-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Social Media Manager screen to configure the plugin

== Usage ==

1. After installation, go to Settings -> Social Media Manager
2. Enter your social media URLs
3. Toggle the platforms you want to display
4. Drag and drop to reorder the icons
5. Choose whether to display icons in header or footer
6. Save your changes

To display the icons manually in your theme or posts, use the shortcode:
[social_media_icons]

== Frequently Asked Questions ==

= Can I control where the icons appear? =
Yes, you can choose between header and footer placement, or use the shortcode [social_media_icons] to place them anywhere.

= Can I change the order of the icons? =
Yes, simply drag and drop the icons in the admin panel to reorder them.

= Can I add custom social media platforms? =
The current version supports the most popular platforms. Custom platforms will be added in future updates.
